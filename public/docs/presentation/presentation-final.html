STAGE 1MISSION BRIEFING

01. 프로젝트 개요

프로젝트명

Trend-Hybrid Admin (부제 : NAZAKZON (나의 작은 아마존))

한 줄 소개

틱톡 바이럴로 검증된 상품을 아마존 데이터로 교차 확인하고, 타오바오 원가로 소싱하여 Shopify에 등록하는 트렌드 헌팅 자동화 솔루션.

선정 배경 (문제 정의)

  • Workflow Fragmentation: 소싱부터 Shopify 등록까지의 워크플로우가 수동 단계로 분산되어 있어, 각 단계마다 사용자 개입이 필요함
  • Automation Gap: 트렌드 상품 데이터 수집, 원가 확인, 가격 계산, Shopify API 연동까지의 전 과정이 수동 처리되어 Time-to-Market이 지연됨
  • Market Logic: "틱톡 조회수 급상승 → 아마존 키워드 검색량 폭발"이라는 필승 공식 존재
  • Sourcing Gap: 트렌드 상품(미국)과 제조 원가(중국) 사이의 정보 비대칭
  • Global Sales Barrier: 글로벌 세일즈 진입을 위한 기술적/운영적 장벽이 높아 1인 셀러의 해외 시장 진입이 어려움. 외화 획득 기회 제한

기획 의도

소싱부터 Shopify 등록까지 클릭 한 번에 완전 자동화를 통해 1인 셀러의 Time-to-Market을 최소화하고, 순차 처리 시스템으로 안정적인 워크플로우 자동화 제공. 글로벌 세일즈의 장벽을 낮춰 해외 시장 진입을 용이하게 하고, 외화 획득 기회를 확대하여 해외 수익 창출을 지원.
