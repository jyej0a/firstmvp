## 슬라이드 3: 프로젝트 개요 - 주제 및 선정 배경

### 프로젝트명
**Trend-Hybrid Admin (트렌드-하이브리드 어드민)**

### 한 줄 소개
틱톡 바이럴로 검증된 상품을 아마존 데이터로 교차 확인하고, 타오바오 원가로 소싱하여 Shopify에 등록하는 **트렌드 헌팅 자동화 솔루션**

### 선정 배경

**문제 정의:**
- **Workflow Fragmentation**: 소싱부터 Shopify 등록까지의 워크플로우가 수동 단계로 분산되어 있어, 각 단계마다 사용자 개입이 필요함
- **Automation Gap**: 트렌드 상품 데이터 수집, 원가 확인, 가격 계산, Shopify API 연동까지의 전 과정이 수동 처리되어 Time-to-Market이 지연됨
- **Market Logic**: "틱톡 조회수 급상승 → 아마존 키워드 검색량 폭발"이라는 필승 공식 존재
- **Sourcing Gap**: 트렌드 상품(미국)과 제조 원가(중국) 사이의 정보 비대칭
- **Global Sales Barrier**: 글로벌 세일즈 진입을 위한 기술적/운영적 장벽이 높아 1인 셀러의 해외 시장 진입이 어려움. 외화 획득 기회 제한

**기획 의도:**
소싱부터 Shopify 등록까지 **클릭 한 번에 완전 자동화**를 통해 1인 셀러의 Time-to-Market을 최소화하고, 순차 처리 시스템으로 안정적인 워크플로우 자동화 제공. **글로벌 세일즈의 장벽을 낮춰 해외 시장 진입을 용이하게 하고, 외화 획득 기회를 확대하여 해외 수익 창출을 지원.**

## 슬라이드 5: 프로젝트 개요 - 프로젝트 내용

### 목적
소싱부터 Shopify 등록까지 **클릭 한 번에 완전 자동화**를 통해 Time-to-Market을 최소화하고, 순차 처리 시스템으로 안정적인 워크플로우 자동화 제공

### 진행 일정
- **Phase 0**: 인프라 세팅 (2025/12/03 ~ 2025/12/12)
- **Phase 2**: 코어 개발 (2025/12/13 ~ 2025/12/29)
- **Phase 3**: 테스트 & 런칭 (2025/12/30 ~ 2026/01/09)
- **🚀 MVP 1.0 런칭**: 2026/01/10

### 구체적인 실행 내역

**MVP 1.0 핵심 기능:**
1. **순차 처리 시스템**: 아마존 상품 데이터 수집 (하루 최대 1000개, 1분당 1개씩) 및 자동 등록
2. **One-Click Automation**: 소싱부터 Shopify 등록까지 클릭 한 번에 완전 자동화
3. **금지어 자동 필터링**: 브랜드명 등 금지어 포함 시 수집 단계에서 자동 차단
4. **US 타입 가격 자동 계산**: 마진율 기반 실시간 가격 계산 (API 기반)
5. **Shopify API 연동**: 순차 처리 방식으로 안정적인 일괄 등록 (1개 수집 → 등록 → 다음 수집)
6. **타오바오 이미지 검색 브릿지**: 원가 확인 지원 (수동 검색)

## 슬라이드 7: 프로젝트 개요 - 기대 효과

### 비즈니스 실무 활용성

**1. 시간 절약**
- 기존: 트렌드 발견 → 수동 검색 → 데이터 수집 → 가격 계산 → 등록 (며칠 소요)
- 개선: 자동화된 워크플로우로 **몇 시간 내 완료**

**2. 수익성 향상**
- 마진율 자동 계산으로 최적 가격 설정
- 빠른 시장 진입으로 경쟁 우위 확보

**3. 확장성**
- 하루 최대 1000개 상품 수집 가능 (약 16.7시간 소요)
- 안정적인 수집 속도 (1분당 1개)로 Bot Detection 위험 최소화

**4. 글로벌 세일즈 접근성**
- 글로벌 세일즈의 기술적/운영적 장벽을 낮춰 1인 셀러도 해외 시장 진입 가능
- 외화 획득 기회 확대 및 해외 수익 창출 지원
- Shopify 글로벌 플랫폼을 통한 자동화된 해외 판매 파이프라인 구축

### 프로젝트 산출물의 효용성

✅ **실제 운영 가능한 MVP**: Shopify 연동 완료, 실제 상품 등록 테스트 성공  
✅ **확장 가능한 아키텍처**: 순차 처리 시스템으로 대규모 수집 지원  
✅ **사용자 친화적 UI**: 직관적인 대시보드 및 실시간 진행 상황 표시  
✅ **글로벌 세일즈 민주화**: 기술적 장벽 제거로 1인 셀러의 해외 시장 진입 용이

// 가격 계산 로직
export function calculateSellingPriceUS(
  amazonPrice: number,
  marginRate: number
): number {
  if (amazonPrice <= 0 || marginRate < 0 || marginRate > 100) {
    throw new Error('Invalid input parameters');
  }
  return amazonPrice * (1 + marginRate / 100);
}
