>
> 예를 들어, '목 교정기'가 뜨고 있다면, 아마존에서 'Neck Traction Device'로 검색하시면 됩니다.

**나레이션 (계속)**:

> "입력창에 키워드나 URL을 입력하시면, 시스템이 자동으로 아마존 검색 URL을 생성합니다.
>
> 그 아래에는 'Target Count' 입력란이 있습니다. 기본값은 1000개이며, 원하시는 개수를 입력하실 수 있습니다.

> 그리고 'Collect & Sync' 모드와 'Collect Only' 모드를 선택하실 수 있습니다.
>
> - Collect & Sync: 수집과 동시에 Shopify에 자동 등록

>
> 처음 사용하시는 분은 'Collect Only' 모드로 시작하시는 것을 권장합니다."

### [2:30-4:00] 3단계: 수집 시작 및 진행 상황 확인

**나레이션**:

> "모든 설정이 완료되셨다면, 'Start' 버튼을 클릭하시면 됩니다."

**[화면]**:

- Start 버튼 클릭
- 진행 상황 바가 나타나는 모습

**나레이션 (계속)**:

> "수집이 시작되면, 화면에 진행 상황이 실시간으로 표시됩니다.
>
> 시스템은 1분당 1개씩 안정적으로 수집하며, 이는 Bot Detection을 회피하기 위한 안전한 속도입니다.
>
> 진행 상황 표시 영역에는 크게 세 가지 통계가 **큰 숫자로** 표시됩니다:
>
> **첫 번째, 수집된 상품 개수**: '5 / 100'과 같이 현재 수집된 개수와 목표 개수가 표시됩니다. 이 숫자는 **5초마다 자동으로 업데이트**되며, 상품이 하나씩 수집될 때마다 숫자가 **1씩 증가**합니다. 예를 들어, '5 / 100'에서 '6 / 100'으로, '7 / 100'으로 하나씩 올라가는 모습을 보실 수 있습니다.
>
> **두 번째, 실 소요시간**: 수집이 시작된 후 실제로 경과한 시간이 표시됩니다. '1분 30초', '2시간 15분'과 같이 실시간으로 업데이트되며, **1초마다 갱신**됩니다.
>
> **세 번째, 예상 남은 시간**: 남은 상품 수를 기준으로 예상 소요 시간이 계산되어 표시됩니다. 예를 들어, 100개 중 5개를 수집했다면, 남은 95개를 수집하는 데 약 95분이 소요될 것으로 예상됩니다.
>
> 그 아래에는 **진행률 바**가 있으며, 퍼센트로 진행률이 표시됩니다. 예를 들어, 100개 중 50개를 수집했다면 50%로 표시되고, 진행률 바가 절반까지 채워집니다.
>
> 그리고 **성공 개수**와 **실패 개수**도 별도로 표시되어, 수집이 성공적으로 완료된 상품과 실패한 상품의 개수를 한눈에 확인하실 수 있습니다.
>
> 예를 들어, 100개를 수집하려면 약 100분, 즉 1시간 40분 정도 소요되며, 이 모든 정보가 **실시간으로 업데이트**되어 표시됩니다."

**[화면]**:

- 진행 상황 바의 실시간 업데이트
- **중요**: 숫자가 하나씩 증가하는 모습을 클로즈업으로 보여주기
  - '5 / 100' → '6 / 100' → '7 / 100'으로 숫자가 변경되는 모습
  - 실 소요시간이 '1분 5초' → '1분 6초' → '1분 7초'로 증가하는 모습
  - 예상 남은 시간이 '95분' → '94분' → '93분'으로 감소하는 모습
  - 진행률 바가 점진적으로 채워지는 모습
  - 성공/실패 개수가 증가하는 모습

**나레이션 (계속)**:

> "수집이 진행되는 동안, 화면 하단의 '오늘 수집 현황' 섹션에서 수집된 상품 목록을 확인하실 수 있습니다.
>
> 각 상품은 ASIN, URL, 그리고 완료 상태가 표시됩니다.
>
> - 파란색 로딩 아이콘: 수집 중 (draft 상태)
> - 초록색 체크: Shopify 등록 완료 (uploaded 상태)
> - 빨간색 X: 등록 실패 (error 상태)
>
> 수집이 완료되면, 'ProductList에서 전체 상품 보기' 버튼을 클릭하여 상세 정보를 확인하실 수 있습니다."

### [4:00-5:30] 4단계: 상품 목록 확인 및 원가 확인

**나레이션**:

> "Product List 페이지로 이동하시면, 수집된 모든 상품을 확인하실 수 있습니다."

**[화면]**:

- Product List 페이지
- 상품 목록 테이블

**나레이션 (계속)**:

> "각 상품에는 다음 정보가 표시됩니다:
>
> - 체크박스: 등록할 상품 선택
> - 이미지: 상품 이미지 (클릭 가능)
> - 제목: 상품명
> - ASIN: 아마존 상품 고유 번호
> - Type: 소싱 타입 (US 또는 CN)
> - Cost: 원가 (CN 타입일 경우 입력 필요)
> - Price: 판매가 (자동 계산)
> - Status: 현재 상태
>
> 가장 중요한 것은 상품 이미지를 클릭하는 것입니다. 이미지를 클릭하시면, 타오바오 이미지 검색 결과 페이지가 새 탭으로 열립니다."

**[화면]**:

- 상품 이미지 클릭
- 타오바오 이미지 검색 결과 페이지가 열리는 모습

**나레이션 (계속)**:

> "타오바오에서 동일하거나 유사한 상품을 찾으시면, 원가를 확인하실 수 있습니다.
>
> 예를 들어, 타오바오에서 $3에 판매되는 상품을 발견하셨다면, Product List로 돌아와서 해당 상품의 'Type'을 'CN'으로 변경하고, 'Cost' 입력란에 3을 입력하시면 됩니다.
>
> 원가를 입력하시면, 시스템이 자동으로 마진율을 적용하여 판매가를 계산합니다."

**[화면]**:

- Type을 CN으로 변경
- Cost 입력란에 원가 입력
- Price가 자동으로 계산되는 모습

---

### [5:30-7:00] 5단계: 마진율 설정 및 가격 계산

**나레이션**:

> "마진율은 Margin Rate 페이지에서 설정하실 수 있습니다."

**[화면]**:

- Margin Rate 페이지로 이동
- 마진율 입력 화면

**나레이션 (계속)**:

> "기본 마진율은 30%로 설정되어 있으며, 원하시는 마진율을 입력하시면 됩니다.
>
> 예를 들어, 40% 마진율을 원하신다면, 40을 입력하시면 됩니다.
>
> 마진율을 변경하시면, Product List의 모든 상품 가격이 자동으로 재계산됩니다."

**[화면]**:

- 마진율 40% 입력
- Product List로 돌아가서 가격이 업데이트되는 모습

**나레이션 (계속)**:

> "가격 계산 방식은 소싱 타입에 따라 다릅니다.
>
> - US 타입: 아마존 가격 × (1 + 마진율 / 100)  
>   예: 아마존 가격 $29, 마진율 40% → 판매가 $40.6
>
> - CN 타입: (원가 + 배송비 + 관세) / (1 - 마진율 / 100)  
>   예: 원가 $3, 배송비 $5, 관세 $2, 마진율 40% → 판매가 $16.67
>
> 이렇게 계산된 가격으로 Shopify에 등록됩니다."

---

### [7:00-8:30] 6단계: 상품 선택 및 Shopify 일괄 등록

**나레이션**:

> "원가 확인과 가격 계산이 완료되셨다면, 이제 Shopify에 등록할 상품을 선택하시면 됩니다."

**[화면]**:

- Product List 페이지
- 여러 상품의 체크박스 선택

**나레이션 (계속)**:

> "등록하고 싶은 상품의 체크박스를 선택하시면, 상단에 '선택 등록 (N)' 버튼이 활성화됩니다.
>
> 주의하실 점은, CN 타입 상품의 경우 원가(Cost)가 입력되어 있어야 한다는 것입니다. 원가가 입력되지 않은 상품이 선택되어 있으면, 경고 메시지가 표시되고 등록이 진행되지 않습니다."

**[화면]**:

- 원가 미입력 상품 선택 시 경고 메시지
- 원가 입력 후 다시 선택

**나레이션 (계속)**:

> "모든 조건이 충족되셨다면, '선택 등록' 버튼을 클릭하시면 됩니다."

**[화면]**:

- 선택 등록 버튼 클릭
- 로딩 상태 표시

**나레이션 (계속)**:

> "등록이 진행되면, 각 상품이 순차적으로 Shopify에 전송됩니다.
>
> 시스템은 다음과 같은 정보를 Shopify에 등록합니다:
>
> - 상품명 (제목)
> - 상품 설명 (HTML 형식)
> - 이미지 (최대 10개)
> - 가격 (계산된 판매가)
> - 옵션 (색상, 크기 등)
> - 카테고리
> - 브랜드
>
> 등록이 완료되면, 성공 메시지와 함께 등록된 상품 개수가 표시됩니다."

**[화면]**:

- 등록 진행 중 로딩
- 성공 메시지 표시
- 상품 상태가 'uploaded'로 변경되는 모습

---

### [8:30-9:30] 7단계: 등록 결과 확인 및 History 조회

**나레이션**:

> "등록이 완료되면, Product List에서 상품 상태가 'uploaded'로 변경된 것을 확인하실 수 있습니다."

**[화면]**:

- Product List에서 상태 변경 확인
- uploaded 상태의 상품들

**나레이션 (계속)**:

> "또한, History 페이지에서 수집 및 등록 이력을 확인하실 수 있습니다."

**[화면]**:

- History 페이지로 이동
- Scraping History 탭과 Upload History 탭

**나레이션 (계속)**:

> "History 페이지에는 두 개의 탭이 있습니다:
>
> - Scraping History: 수집 작업 이력
>
>   - 날짜, 키워드/URL, 상태, 진행률, 성공/실패 개수, 소요 시간 등이 표시됩니다.
>
> - Upload History: Shopify 등록 이력
>   - 등록 날짜, 상품명, ASIN, 상태 등이 표시됩니다.
>
> 이를 통해 언제 어떤 상품을 수집하고 등록했는지 한눈에 확인하실 수 있습니다."

### [9:30-10:30] 8단계: 상품 상세 편집 (Editor)

**나레이션**:

> "등록 전에 상품 정보를 수정하고 싶으시다면, Editor 페이지를 사용하실 수 있습니다."

**[화면]**:

- Editor 페이지
- 상품 선택 및 편집 화면

**나레이션 (계속)**:

> "Editor에서는 다음 정보를 수정하실 수 있습니다:
>
> - 상품명
> - 상품 설명
> - 원가 (Cost)
> - 마진율
> - 판매가 (자동 계산)
>
> 수정이 완료되면, 'Shopify 등록' 버튼을 클릭하여 개별적으로 등록하실 수 있습니다."

**[화면]**:

- 상품 정보 수정
- Shopify 등록 버튼 클릭

### [10:30-11:00] 9단계: Collect & Sync 모드 사용법

**나레이션**:

> "이제 Collect & Sync 모드에 대해 설명드리겠습니다."

**[화면]**:

- Start 페이지로 돌아가기
- Collect & Sync 모드 선택

**나레이션 (계속)**:

> "Collect & Sync 모드를 선택하시면, 수집과 동시에 Shopify에 자동으로 등록됩니다.
>
> 이 모드는 다음과 같이 동작합니다:
>
> 1. 상품 1개 수집
> 2. 즉시 Shopify에 등록
> 3. 다음 상품 수집
> 4. 반복
>
> 이렇게 하면 수집이 완료되는 대로 모든 상품이 Shopify에 등록되어, 더욱 빠르게 판매를 시작하실 수 있습니다.
>
> 다만, 원가 확인이나 가격 조정이 필요한 경우에는 Collect Only 모드를 사용하시는 것을 권장합니다."

## 📋 영상 제작 체크리스트

### 필수 촬영 장면

- [ ] 로그인 화면 및 대시보드 접속
- [ ] Start 페이지의 Quick Links 클릭
- [ ] 키워드/URL 입력 및 수집 시작
- [ ] **진행 상황 바 실시간 업데이트 (중요)**
  - [ ] 수집된 상품 개수가 하나씩 증가하는 모습 (클로즈업)
  - [ ] 실 소요시간이 1초마다 업데이트되는 모습
  - [ ] 예상 남은 시간이 감소하는 모습
  - [ ] 진행률 바가 점진적으로 채워지는 모습
  - [ ] 성공/실패 개수가 증가하는 모습
- [ ] Product List에서 상품 이미지 클릭 (타오바오 검색)
- [ ] 원가 입력 및 가격 자동 계산
- [ ] 마진율 설정
- [ ] 상품 선택 및 일괄 등록
- [ ] History 페이지 조회
- [ ] Editor 페이지 상품 편집

### 화면 전환 효과

- 화면 전환 시 부드러운 페이드 효과
- 중요한 버튼 클릭 시 확대 효과
- **진행 상황 업데이트 시 숫자 카운트 애니메이션 (중요)**
  - 수집된 상품 개수가 증가할 때 숫자가 하나씩 올라가는 모습을 명확하게 보여주기
  - 예: '5 / 100' → '6 / 100' → '7 / 100'으로 숫자가 변경되는 장면을 클로즈업으로 촬영
  - 실 소요시간이 1초마다 업데이트되는 모습
  - 예상 남은 시간이 감소하는 모습
  - 진행률 바가 점진적으로 채워지는 모습 (0% → 10% → 20% ...)
  - 성공/실패 개수가 증가하는 모습

- 각 단계별 자막 표시
- 중요한 버튼/입력란에 하이라이트

- 명확하고 차분한 목소리
